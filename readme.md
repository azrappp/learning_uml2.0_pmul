# Sequence Diagrams Repository

This repository contains **sequence diagrams** implemented using **PlantUML**, following the **UML 2.0** standard.

## Overview

- All diagrams are structured for clarity and maintainability.
- Each `.puml` file represents a distinct interaction scenario.
- Designed for documentation, system analysis, and software design purposes.

## Usage

1. Open any `.puml` file in a PlantUML-compatible editor or viewer.
2. Render the diagram to visualize the sequence of interactions.

## Contribution

- Ensure diagrams comply with **UML 2.0** conventions.
- Maintain clear naming and structured frames.
- Submit changes via pull request for review.
